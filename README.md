# ProyectoXML

Este proyecto es un cuestionario bilingüe (español e inglés) sobre coches, hecho con XML, JavaScript, y AJAX.

## ✅ Características
- XML validado con DTD
- Selector de idioma
- Contador de tiempo total
- Puntuación al final del test
- Interfaz simple y clara
